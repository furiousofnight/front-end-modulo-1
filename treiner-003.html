O que é um parágrafo?

No HTML, o parágrafo é representado pela tag <p>. Ele serve para separar blocos de texto e organizar o conteúdo da página.

Exemplo de um parágrafo simples:

<p>Este é um parágrafo no HTML.</p>

Como ele aparece visualmente:

Este é um parágrafo no HTML.

Quebras de linha

Quando você quer iniciar uma nova linha sem criar um novo parágrafo, você usa a tag <br>. Ela não precisa de fechamento.

Exemplo de quebra de linha:

<p>Linha 1<br>Linha 2<br>Linha 3</p>

Visualização do exemplo acima:

Linha 1
Linha 2
Linha 3

Separando seções com <hr>

A tag<hr>cria uma linha horizontal na página, útil para separar seções ou tópicos de forma visual.

Exemplo: <hr>

Exemplo Visual

O conteúdo acima da linha está separado do conteúdo abaixo por uma linha branca.

Resumo

1. <p> cria um parágrafo, separando blocos de texto.

2. <br> cria uma quebra de linha dentro de um parágrafo.

3. <hr> cria uma linha horizontal para separar seções.

Estas três tags ajudam a organizar o conteúdo de forma clara e legível.
