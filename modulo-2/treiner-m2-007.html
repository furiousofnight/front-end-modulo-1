Introdução ao Desafio

Nesta aula você irá criar um site real simples, aplicando os conceitos de HTML e CSS que aprendeu nas aulas anteriores. O objetivo é compreender como estruturar um documento HTML, estilizar elementos com CSS e organizar o conteúdo de forma clara e harmoniosa.

Durante o desafio, você terá liberdade para experimentar cores, fontes e layouts, criando seu próprio estilo visual.

Dicas Importantes

  • Mantenha seu HTML semântico e organizado para facilitar futuras alterações.

  • Experimente combinar cores e sombras para tornar o site mais atrativo.

  • Centralize conteúdos importantes e use espaçamentos consistentes.

  • Teste seu site em diferentes tamanhos de tela para garantir responsividade.

  • Lembre-se: menos é mais. Evite poluir visualmente a página com elementos desnecessários.

  • Utilize comentários no código para marcar seções e facilitar a manutenção.
